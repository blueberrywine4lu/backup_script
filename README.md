# Backup Script

Script Python per il backup automatico di file su Linux.

## Cosa fa
- Copia i file da una cartella di origine a una di destinazione
- Aggiunge data e ora al nome di ogni file copiato
- Salta automaticamente le cartelle
- Permette di avere più versioni dello stesso file nel tempo

## Tecnologie usate
- Python 3
- Linux (Debian)

## Come si usa
1. Modifica `origine` e `destinazione` nel file con i tuoi percorsi
2. Esegui con `python3 backup.py`
